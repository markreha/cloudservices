IoT REST API

The IoT REST API is written in JAX-RS using the Spring Framework. The API's provides the following services:

  • 1. Save Weather Data. Intended for consuming IoT Device Applications to save their data.
  • 2. Query Weather Data by Device. Intended for consuming IoT Reporting Application to retrieve Weather data.
  • 3. Query Weather Data by Date Range. Intended for consuming IoT Reporting Application to retrieve Weather data.


The IoT REST Service acts as Facade over the IoT Weather Service and IoT Weather DAO.


The IoT REST Service implements HTTP Basic Authentication over HTTPS for security.
